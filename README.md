# bacillus_c64
Jump and run game for the C64.
Control a bacillus to hunt for food.

Technical facts:
* 2-way side scrolling game
* parallax scrolling
* animated background and level data
* animated characters

Planned:
* Intro, Outtro
* Highscore handling
* More levels
* More level graphics
* More enemies
* Boni
* Music/Sfx
